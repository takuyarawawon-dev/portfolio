# Portfolio – たくやらわをん@dev

このリポジトリは、個人ポートフォリオサイトのソースコードです。  
HTML / CSS / JavaScript を使い、GitHub Pages で公開しています。
フレームワークを使わず、構造や内容そのものに集中できる構成を選びました。

## コンセプト

インターネット上にエンジニアとしての人格を作ることを目的に、
そのきっかけとなった記事を読み、各種アカウントを作成しました。

その最初の成果物として、自身を表現し、活動リンクを集約できる
ポートフォリオサイトが適していると考え、このサイトを作成しました。

## このサイトで表現したいこと

- クリーンな構造（セマンティックHTML）
- シンプルで読みやすいUI
- 後から拡張できる最小構成

## 使用技術

- HTML5
- CSS3
- JavaScript（最小限）
- GitHub Pages（ホスティング）
